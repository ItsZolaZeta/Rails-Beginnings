Option 1: Pin your images

Built by: Tatiana Tuor

This is a website heavily inspired by the functionality of pinterest. You can create pins which consist of an image, title and tags. You can log in or sign up with an email, no password required. When logged in, you can collect/save pins that you really enjoy under a personal pin list. When logged in, you can add comments to pins to express what you think about them. You can search for pins of a certain type using the search field, which will return you results where the search term was in either the title or tags.
